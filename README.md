# Customer Churn Prediction System



## Problem Statement

Marketing is losing customers to competitors. Acquiring new customers costs 5x more than retaining existing ones. We need a system to identify customers at risk of churning \*before\* they leave.



## Goals

1. Predict the probability of a customer churning (0-100%).

2. API that accepts customer data and returns a risk score.

3. Dashboard for the marketing team.



## Success Metric

- Primary: Recall (We must catch at least 80% of churners).

- Secondary: Precision (Don't spam too many happy customers)
